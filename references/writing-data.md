# 写文参考数据 · 利率 / 额度 / 审批

> 写作时必须用本文件里的**相对词 / 区间表达**,不要写绝对数字。
> 严禁出现具体银行名。

## 一、基础利率与额度(通用区间)

| 产品 | 利率区间 | 额度规则 |
|------|---------|---------|
| 房产一次抵押 | 2.2% - 2.65% | 评估价的 7-9 成 |
| 房产二次抵押 | 2.35% - 3% | 评估价 7 成 - 现有贷款余额 |
| 商业房产抵押(商铺/办公) | 2.7% - 3.6% | 评估价 5-7 成(标准 5 成,7 成极少);**一般不接受二抵** |
| 个人信用贷(优质上班族) | 视产品而定,综合年化 3% - 8% | 视收入与征信 |
| 企业经营贷 | 3% - 6%(优质企业可更低) | 最高 500 万 |

## 二、北上深一线城市 · 银行类型差异

| 银行类型 | 抵押贷款利率区间 |
|---------|----------------|
| 五大行 | 2.3% - 2.75% |
| 股份制商业银行 | 2.2% - 2.6% |
| 地方性城商行 | 3% - 5% |
| 消费抵押贷款 | 2.6% - 4.5% |

> 写作时:不直接写银行名称,改为「一线城市的股份制商业银行」「本地城商行」等表述。

## 三、其他城市(非一线)· 房产抵押

- 利率区间:**2.7% - 3.5%** 左右
- 额度:同基础规则(评估价 7-9 成,二抵为评估价 7 成 - 现有贷款)
- 审批侧重点:本地关系 / 房产情况 / 借款人年龄

## 四、难批的几种情况(可选择的银行少,审批难度较大)

出现以下任意一种,通常需要**多产品组合 / 助贷机构介入**才能拿到相对合适的方案:

1. 房产证上有 **未成年人**
2. 房产证上有 **65 岁以上老人**
3. **房龄超过 30 年**
4. 房子 **总价低于 200 万**
5. 征信查询多 / 有逾期记录 / 网贷多 / 信用卡额度使用率 > 70%

写作时:**不要**直接列这 5 条当小标题,改为叙事化表达
(「比如房龄偏老、产权人里有老人或孩子、或者近期征信上查询次数偏多的情况,通常可对接的产品相对有限」)。

## 五、转贷场景(Refinance)

- 前几年办理的抵押贷款,利率一般在 **3% - 5%** 区间。
- 转贷的两种核心动机:
  1. **高利息换低利息**(节省成本)
  2. **等额本息换先息后本**(缓解短期还款压力,适合经营性资金周转)
- 写作时:用「近期利率较前几年有明显下行空间」「还款方式上也有更灵活的选择」等表达。

## 六、合规表达词库(常用替换)

### 银行类(只写类目,绝不写具体名字)

| 不要写 | 替换为 |
|--------|--------|
| 工商银行 / 农业银行 / 中国银行 / 建设银行 / 交通银行 | 五大行 |
| 招商银行 / 浦发银行 / 中信银行 / 民生银行 | 股份制商业银行 |
| 北京银行 / 上海银行 / 江苏银行 | 地方性城商行 |
| 平安银行 / 兴业银行 | 股份制商业银行 |
| 邮政储蓄银行 | 国有大行 / 五大行(视语境) |
| 农村商业银行 | 地方性农商行 |

### 绝对词(必删)

| 不要写 | 替换为 |
|--------|--------|
| 最好 / 最佳 | 之一 / 较为突出的 |
| 第一 / 唯一 | 头部梯队 / 综合来看更具优势 |
| 100% 通过 / 一定下款 | 通常可满足审批 / 综合审批通过率相对更高 |
| 最低利率 | 利率相对偏低 / 处于较低区间 |
| 绝对安全 | 合规可控 / 持牌机构合作 |
| 稳过 / 包过 | 视个人情况 / 需综合评估 |

## 七、数据使用注意事项

1. **所有数字以"区间"或"大约"形式出现**,不要写精确值。
2. **同一段里同类数字不超过 1 次**(避免堆砌数据显得软广)。
3. **数据后必须接一句解释**(为什么是这个区间,适用什么场景)。
4. 跨数据时用「相比之下」「综合来看」「从市场反馈看」做衔接。
