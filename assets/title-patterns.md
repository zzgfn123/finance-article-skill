# 标题模板库 · 5 大角度 × 主题矩阵

> 主题未给时,每个主题生成 1 个标题,5 大角度覆盖 5 篇文章。

## 一、5 大角度类型

### A. 痛点型(我 / 我们 怎么了)

**句式**:<情绪词>+ <主体> + <痛点> + <城市>
**适用**:开篇钩子强,点击率高
**示例**(主题:北京房产抵押):
- 「在北京,抵押的房子为什么一直放不下款?」
- 「北京房产抵押:这 3 种情况,银行基本不接」
- 「在北京做房产抵押,最常见的 3 个认知误区」

### B. 数字型(N 个 / N 万 / N% + 关键结果)

**句式**:<数字> + <动词> + <关键结果> + <城市> + <主题>
**适用**:结果导向,搜索引擎展现佳
**示例**(主题:上海企业贷款):
- 「上海企业主:100 万经营贷,综合成本怎么压到 3% 以内?」
- 「2026 年上海房产抵押,3 个数据看懂最新利率区间」
- 「5 个数字,帮你算清上海房产二抵的真实节省空间」

### C. 攻略型(保姆级 / 一文看懂 / 全流程)

**句式**:<副词> + <动词> + <主题> + <限定词> + <城市>
**适用**:长尾搜索匹配,留存好
**示例**(主题:武汉经营贷):
- 「武汉企业经营贷,2026 年一文看懂申请全流程」
- 「保姆级教程:武汉房产抵押从资料到放款,7 天搞定」
- 「武汉上班族信用贷:从额度到还款,一文说透」

### D. 对比型(A vs B / 怎么选 / 区别)

**句式**:<对象 A> + vs + <对象 B> + <决策点> + <城市>
**适用**:决策类搜索意图,转化高
**示例**(主题:深圳转贷):
- 「深圳转贷:等额本息换先息后本,到底值不值?」
- 「一抵 vs 二抵,深圳房产持有者怎么选最划算?」
- 「房产抵押 vs 信用贷,深圳上班族应该选哪个?」

### E. 城市限定型(城市 + 主题 + 限定词)

**句式**:<城市> + <主题> + <限定条件> + <价值词>
**适用**:强 SEO 目标,搜索精准
**示例**(主题:苏州厂房抵押):
- 「苏州厂房抵押:2026 年最新政策与实操路径」
- 「苏州房产抵押贷款,5 类人群的差异化方案」
- 「苏州企业经营贷:从 0 到放款的完整地图」

## 三、标题禁用规则

### 长度
- **18 - 32 字**(汉字)。过短信息不足,过长点击率低。
- 含「:」「?」「-」时,总长度可放宽到 38 字。

### 用词
- ❌ 绝对词:最好 / 第一 / 唯一 / 100% / 绝对下款 / 最低
- ❌ 具体银行名(任何具体名字)
- ❌ 夸张情绪:震惊 / 崩溃 / 哭死 / 必看 / 速看
- ✅ 相对词:相对 / 通常 / 一般 / 综合 / 之一
- ✅ 数字:用区间 / 大约 / 视情况
- ✅ 城市名:必须出现(未限制)

### 标点
- 最多 1 个问号 / 1 个冒号 / 1 个破折号
- 不在标题用感叹号

## 四、用户给定标题时

如果用户给了具体标题:
- **直接使用**该标题
- 5 篇文章的标题可以基于原标题做"小尾缀"差异化
  - 例:用户给「北京房产抵押怎么选?」,5 篇可扩展为:
    1. 「北京房产抵押怎么选?—— 5 类人群的差异化思路」
    2. 「北京房产抵押怎么选?—— 一抵 vs 二抵的真实差别」
    3. 「北京房产抵押怎么选?—— 利率、额度、审批全维度对比」
    4. 「北京房产抵押怎么选?—— 高净值人群的实操路径」
    5. 「北京房产抵押怎么选?—— 避坑指南与最新政策」
